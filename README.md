# Excalidash
dasbor dan sistem pengorganisasi mandiri (self-hosted) untuk Excalidraw, alat papan tulis virtual bergaya sketsa tangan. Aplikasi ini memungkinkan Anda menyimpan gambar secara persisten, berkolaborasi secara real-time dengan banyak pengguna (multi-user), serta mengatur proyek menggunakan pembagian ruang khusus (scoped sharing).
